## 📖 Giới thiệu

Đây là **portfolio chuyên nghiệp** được thiết kế để showcase kỹ năng, dự án và dịch vụ của tôi với tư cách là một **Software Developer**. Portfolio này được xây dựng với focus vào **modern design**, **responsive layout** và **professional presentation**.

Portfolio được thiết kế để thu hút nhà tuyển dụng và khách hàng tiềm năng, thể hiện khả năng tạo ra các ứng dụng web hiện đại và có tính ứng dụng cao.

### 🎯 Mục tiêu dự án

- ✅ Showcase kỹ năng frontend và backend development
- ✅ Thể hiện các dự án thực tế đã hoàn thành
- ✅ Cung cấp thông tin liên hệ chuyên nghiệp
- ✅ Tạo ấn tượng tốt với nhà tuyển dụng
- ✅ Responsive design cho mọi thiết bị

## 📁 Cấu trúc dự án

```
ndyudev-portfolio-front-end/
├── 📁 assets/
│   ├── 📁 icons/               # Technology icons (React, Java, etc.)
│   ├── 📁 images/              # Project images và backgrounds
│   └── 📁 documents/           # CV và tài liệu khác
├── 📁 css/
│   ├── 📁 sections/            # CSS cho từng section
│   │   ├── header.css          # Header styling
│   │   ├── about-me.css        # About section
│   │   ├── skills.css          # Skills section
│   │   ├── service.css         # Services section
│   │   ├── projects.css        # Projects section
│   │   └── footer.css          # Footer styling
│   ├── main.css                # Main stylesheet
│   ├── responsive.css          # Responsive design
│   ├── animation.css           # Animations và transitions
│   └── theme.css               # CSS variables và colors
├── index.html                  # Main HTML file
└── README.md                   # Project documentation
```

### 📋 Mô tả thư mục:

- **📁 assets/**: Tài nguyên tĩnh (icons, images, documents)

  - `📁 icons/`: SVG icons cho các công nghệ
  - `📁 images/`: Hình ảnh dự án và backgrounds
  - `📁 documents/`: CV và tài liệu PDF

- **📁 css/**: Stylesheets được tổ chức theo module
  - `📁 sections/`: CSS riêng cho từng section
  - `main.css`: Import tất cả CSS files
  - `responsive.css`: Media queries cho mobile
  - `animation.css`: Hover effects và transitions
  - `theme.css`: Color scheme và variables

## 🚀 Hướng dẫn bắt đầu

### 1. Chuẩn bị môi trường

```bash
# Clone repository (nếu có trên GitHub)
git clone https://github.com/ndyudev/portfolio.git

# Hoặc download trực tiếp
# Mở index.html trong browser
```

### 2. Cài đặt công cụ (Optional)

- **VS Code**: [Download](https://code.visualstudio.com/)
- **Live Server Extension**: Cài đặt trong VS Code
- **Git**: [Download](https://git-scm.com/)

### 3. Chạy portfolio

```bash
# Cách 1: Mở trực tiếp
double-click index.html

# Cách 2: Sử dụng Live Server
# Mở VS Code → Open Folder → Right-click index.html → Open with Live Server

# Cách 3: Sử dụng local server
python -m http.server 8000
# Hoặc
npx serve .
```

### 4. Customization

```css
/* Thay đổi màu sắc trong css/theme.css */
:root {
  --p-500: #3b82f6; /* Primary blue */
  --p-700: #1d4ed8; /* Darker blue */
  --n-500: #6b7280; /* Neutral gray */
}
```

## 📝 Tips phát triển portfolio

### 🎯 1. Content Strategy

- **Professional Bio**: Viết giới thiệu ngắn gọn, chuyên nghiệp
- **Project Showcase**: Chọn projects tốt nhất để highlight
- **Skills Presentation**: Focus vào skills relevant với target job
- **Contact Information**: Dễ dàng tìm thấy và liên hệ

### 🎨 2. Design Principles

- **Clean Design**: Minimalist approach
- **Consistent Branding**: Unified color scheme và typography
- **Mobile-First**: Responsive design cho mọi device
- **Fast Loading**: Optimize images và code

### 📱 3. User Experience

- **Easy Navigation**: Clear section separation
- **Call-to-Action**: Prominent buttons cho contact
- **Loading States**: Smooth transitions
- **Error Handling**: Graceful error states

### 🚀 4. Performance Optimization

- **Image Optimization**: Compress images
- **Code Minification**: Minify CSS/JS
- **Lazy Loading**: Load images khi cần
- **Caching**: Browser caching strategies
